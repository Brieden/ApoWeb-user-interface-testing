# Kapersburg Apotheke – Single Page Website

Eine moderne, mobile-first Webseite mit Tailwind CSS (CDN), optimiert für GitHub Pages.

## Schnellstart (lokal testen)

Öffne einfach `index.html` im Browser (Doppelklick oder per Terminal):

```zsh
open index.html
```

## Deployment auf GitHub Pages

1. Neues GitHub-Repository erstellen (öffentlich oder privat mit Pages aktiviert).
2. Diese Dateien committen/pushen (mindestens `index.html`).
3. GitHub Pages aktivieren:
   - Repository → Settings → Pages → **Source**: `Deploy from a branch`
   - **Branch**: `main` (oder `master`), Folder: `/root`.
4. Nach wenigen Minuten ist die Seite unter der GitHub Pages-URL erreichbar.

## Struktur

- `index.html` – Single Page mit Header, Hero, wichtigen Infos, Leistungen, Rezept-Formular (Platzhalter) und Footer.
- `README.md` – Hinweise für Test und Deployment.

- Grün: `#2E7D32`
- Hintergrund hell: `#F7FAFC`

## Inhalte anpassen

- Telefonnummer (`tel:+49603212345`), Adresse und E-Mail (`info@kapersburg-apotheke.de`) im Abschnitt „Adresse & Kontakt“ aktualisieren.
- Impressum/Datenschutz im Footer mit echten Inhalten befüllen.
- Platzhalterbild im Hero ersetzen (z. B. lokales Bild oder lizenzfreie Quelle).

## Hinweis

Das Formular „Rezept vorbestellen“ ist ein Platzhalter. Für produktive Nutzung bitte einen echten Service (z. B. E-Mail/Telefon) anbinden.
